# pii-cleaner

회사 도구 (Jira / Bitbucket / Slack / MS365) 에 흩어진 개인정보를 식별·익명화·삭제하는 자동화 도구.

## 정체성
- 「삭제」는 영구 삭제 + 「익명화 (pseudonymization)」 두 모드 지원
- 회계·계약 관련 데이터는 법정 보관기간 (5~10년) 동안 익명화 유지, 삭제 X
- 모든 작업은 dry-run → 검토 → 실행 3단계
- 모든 대화/산출물은 한국어

## 절대 금지
- 비밀키·토큰 커밋
- Litigation Hold/eDiscovery 보존 정책 우회 (법적 리스크)
- 백업본 직접 삭제 (vendor SLA 영역)
- 회사 컴플라이언스 팀 승인 없이 운영 데이터 변경

## 구조 (예정)
```
pii-cleaner/
├── docs/        # 도구별 삭제 대상·API·법적 고려
├── adapters/    # 도구별 어댑터 (jira/bitbucket/slack/ms365)
├── core/        # 공통 모델·실행 엔진·dry-run·로그
├── policies/    # 시나리오별 정책 (퇴사자/외부고객/기간)
└── config/      # 인증·환경별 설정
```
